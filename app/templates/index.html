About the System

A machine-learning approach to next-day rain prediction, built end-to-end for a BSc research project.

What It Does

The system ranks candidate weather variables by Mutual Information, keeps the ten most predictive of next-day rainfall, and feeds them into a Gaussian Naive Bayes classifier that returns a Rain / No Rain forecast with a calibrated confidence score.

The Dataset

Trained on the weatherAUS dataset — over 145,000 daily observations collected from Australian Bureau of Meteorology weather stations, spanning humidity, temperature, pressure, and wind measurements across roughly a decade.

Research Context

Developed to investigate whether Mutual-Information-guided feature selection can improve the efficiency and interpretability of Naive Bayes weather classification, compared to using every available variable.

The Science

The two ideas the entire system is built on, explained in plain language.

{{ icons.icon_data_scatter() }}

Mutual Information

Mutual Information measures how much knowing one variable reduces uncertainty about another. In plain terms: it asks "if I know today's humidity, how much better can I guess tomorrow's rain?" Features that answer that question well score higher and are kept; the rest are dropped.

{{ icons.icon_math_formula() }}

Gaussian Naive Bayes

Naive Bayes classifies by comparing probabilities: for each class, it assumes every feature follows its own bell-curve (Gaussian) distribution, then combines how likely the observed readings are under each class. "Naive" refers to the simplifying assumption that features are independent given the class — a trade-off that keeps the model fast and surprisingly effective.
